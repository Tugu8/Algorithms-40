Фибоначчийн дарааллыг дараах байдлаар өөрчилсөн:

t₁ = a

t₂ = b

tₙ = tₙ₋₁² + tₙ₋₂ (n > 2)

Таны даалгавар бол n-р гишүүнийг олох юм.

Оролт
Гурван бүхэл тоо a, b, n-ийг нэг мөрөнд зайгаар тусгаарлан өгнө.

0 ≤ a, b ≤ 2

3 ≤ n ≤ 20

Гаралт
tₙ-ийг хэвлэ.

Жишээ

Оролт:

0 1 5  

Гаралт:

5  
Тайлбар:

t₁ = 0

t₂ = 1

t₃ = 1² + 0 = 1

t₄ = 1² + 1 = 2

t₅ = 2² + 1 = 5
